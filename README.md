# SimpleDoc
轻量化在线文档发布服务,支持部署到linux或windwos(.net core 2.2或更高版本)
- 基于markdown语法
- 支持图片和附件上传
- 支持各种代码主题
- 支持一层目录列表
- 支持文章预览列表

应用示例 http://doc.ikende.com/
